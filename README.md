🏪 **Kirana Saver MVP**
A hyperlocal platform to help local shopkeepers sell near-expiry products to nearby customers.

🚀 **Quick Start Guide**
**1. Environment Setup**
First, ensure you have Python installed and set up your project directory.

# Check if Python is installed
python --version

# Create the project folder
mkdir kirana-app
cd kirana-app

# Install necessary libraries
pip install flask flask-cors

**2. Project Structure**
Create a folder named templates inside your project directory and ensure your files are organized as follows:

app.py (The Python Backend)

templates/index.html (The Frontend UI)

**3. Running the Application**
Open in VS Code: * Launch VS Code.

Click File → Open Folder and select the kirana-app folder.

Start the Server: * Open the integrated terminal in VS Code (Ctrl + ~).

**Run the command:**
python app.py

**View the App:**

Open your browser and go to: http://127.0.0.1:5000

**🛠 Tech Stack Used**
Backend: Python (Flask)

Frontend: HTML5, CSS3, JavaScript

API: Flask-CORS (for handling cross-origin requests)
